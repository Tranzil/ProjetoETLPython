# Projeto ETL Python

Java RESTful API como desafio de projeto da trilha Santander 2025 - Ciência de dados com Python.

## Principais Tecnologias
 - **Python**: Utilizei a versão mais recenta da linguagem python, bem como suas bibliotecas pandas e json;
 - **Gemini**: LLM utilizada para a geração dos dados salvos do arquivo CSV usados na etapa de extração;
 - **OpenAI API**: API do Chat GPT usada para enriquecer e processar os dados das interações dos clientes;
